## Overview

Asurad Documentation empowers you to build stunning, interactive technical docs with MDX and modern components. You create pages that engage developers and users through rich visuals, code examples, and guided workflows. This platform handles rendering, hosting, and SEO so you focus on content.

### Quick Start

Follow these steps to launch your first doc site.

<Steps>
  <Step title="Initialize Project" icon="download">
    Create a new repository and add your first MDX file.

````bash
npx create-asurad@latest my-docs-site
cd my-docs-site
npm install
````

  </Step>
  <Step title="Add Content" icon="edit-3">
    Write MDX pages with frontmatter and components.

    ```javascript
    // pages/introduction.mdx
    ---
    title: "Welcome"
    ---
    
    ## Hello World
    Your docs start here.
    ```

  </Step>
  <Step title="Preview Locally" icon="play">
    Run the dev server.

````bash
npm run dev
````

    Open `http://localhost:3000` to see live previews.
  </Step>
  <Step title="Deploy" icon="upload-cloud">
    Push to GitHub for automatic deployment.

````bash
git add .
git commit -m "Initial docs"
git push origin main
````

  </Step>
</Steps>

## Integration Examples

Embed Asurad components in your apps or embed docs in sites.

<CodeGroup tabs="React,Node.js">
  ```tsx
  // React app integration
  import { Callout } from '@asurad/docs-ui';
  
  function DocsEmbed() {
    return (
      <Callout kind="info">
        Embedded docs component.
      </Callout>
    );
  }
  ```
  ```javascript
  // Node.js server-side rendering
  const asurad = require('@asurad/docs');
  
  app.get('/docs/:page', async (req, res) => {
    const html = await asurad.render(req.params.page);
    res.send(html);
  });
  ```
</CodeGroup>
